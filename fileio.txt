
Which header do you need?

    #include <stdio.h>

How to open a (text) file?

    FILE *fopen( const char * filename, const char * mode );
    FILE *fp = fopen("test.txt","w");

    modes
        -r for reading
        -w for writing
        -r+ for both



How to close a file?

    int fclose( FILE *fp );

How to read a single character from a file?

    int fgetc( FILE * fp );

But what if it's at the end of the file?

    returns EOF
